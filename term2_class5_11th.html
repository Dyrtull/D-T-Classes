Step 18: Generating the List (Challenge)

You need to list properties based on the Location in B4.

  • The Logic: You need to find the correct column for "Ubud" (MATCH) and then pull rows 1, 2, 3... (INDEX).
  • Formula Structure:

    =INDEX(DataRange, ROW(A1), MATCH(Location, HeaderRow, 0))

  • Replication: Drag this formula down from B10 to B15. Ensure you used Absolute Referencing ($) correctly!
