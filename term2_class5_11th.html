Part 1: Layout & Formatting (Video 06:00 - 10:00)

  • Open: Week1_Solar_Inventory.csv in Excel.
  • Row 1 Headers: Select A1:G1. Change Orientation to 90 degrees (Vertical Text Up). Apply Wrap Text.
  • Column Widths: Double-click boundaries to "Auto-fit" columns A, C, D, E, F, G. Manually adjust Column B to approx 30 units.
  • Date Format: Select F2:F11. Change format to Long Date.

Part 2: Borders & Visual Style (Video 42:00 - 47:00)

  • Main Borders (A1:G11): Apply a Thick Box Border to the outside. Change Line Color to RED.
  • Inside Grid: Apply All Borders (thin) to the inside. Change Line Color to BLUE.
  • Header Style: Select A1:G1. Fill Color: Yellow. Make text Bold.

Part 3: Calculations & Fill Handle (Video 18:00 - 27:00)

  • Total Value (Column G): In G2, enter a formula to calculate the Value (Unit Cost multiplied by Quantity).
  • Constraint: You MUST use cell references.
  • Fill Handle: Drag the formula down from G2 to G11.

Part 4: Functions & Statistics (Video 25:00 - 33:00)

Locate the summary section starting at Row 13.

  • Total Stock (E13): Use the SUM function to calculate the total items in the Quantity column.
  • Avg Cost (E14): Use the AVERAGE function to calculate the average of the Unit Costs.
  • Highest Cost (E15): Use the MAX function to find the highest Unit Cost.
  • Lowest Cost (E16): Use the MIN function to find the lowest Unit Cost.
  • Price Spread (E17): Calculate the difference between the Highest and Lowest cost (Max - Min).

Part 5: Advanced Formatting (Video 34:00 - 37:00)

  • Non-Contiguous Selection: Select D2:D11. Hold Ctrl. Select G2:G11. Hold Ctrl. Select E14:E17.
  • Action: With all three ranges selected, format as Currency ($) with 2 decimal places.
  • Save: As Week1_Homework_YOURNAME.xlsx.
