Term 1

Week 1: Introduction to Computer Systems (Ch 1.1-1.2)

Link to Class Presentation

Theory Concepts & Definitions:

Hardware vs. Software: Differentiate between the physical components of a computer system and the programs that run on them.

Internal and External Hardware Components: Identify and describe the functions of components like CPU, Motherboard, RAM, ROM, keyboard, mouse, etc.

RAM vs. ROM: Explain the difference between volatile (RAM) and non-volatile (ROM) memory.

In-Class Exercises:

"Hardware vs. Software Sort" (Group Activity): Categorize cards with names of various items as either hardware or software.

"Component Identification Challenge" (Hands-on/Visual): Identify and describe the function of components in a disassembled computer or diagram.

Basic Computer Familiarization (Practical): Practice logging in, navigating the desktop, and using a simple application.

Possible Homework:

Research & List: Find and list 3 examples of application software and 3 examples of system software.

Reflection: Explain why understanding the difference between hardware and software is important.

Week 2: Operating Systems & Types of Computers (Ch 1.3-1.5)

Link to Class Presentation

Theory Concepts & Definitions:

Operating Systems: The master program that manages all computer hardware and software resources and provides common services for computer programs.

CLI vs. GUI: Differentiate between Command Line Interface and Graphical User Interface.

Types of Computers: Identify the characteristics and uses of desktop, mobile, and emerging computers.

In-Class Exercises:

"OS Function Brainstorm" (Pair Work): Brainstorm and list as many functions of an operating system as you can think of.

"Device Dilemma" (Group Scenario): Recommend the most suitable computer for a given scenario and justify your choice.

GUI Navigation Practice (Practical): Practice basic file management using a GUI.

Possible Homework:

Comparison Table: Create a table comparing GUI and CLI with at least 3 advantages and 3 disadvantages for each.

Scenario Response: Advise a school on purchasing new devices for students with different needs.

Week 3: Input, Output, and Storage Devices (Ch 2-3)

Link to Class Presentation

Theory Concepts & Definitions:

Input, Output, and Direct Data Entry: Identify and describe various devices for inputting and outputting information, as well as direct data entry devices like magnetic stripe readers and barcode scanners.

Storage Media: Differentiate between magnetic (HDD), optical (CD/DVD), and solid-state (SSD) storage, and discuss their pros and cons.

In-Class Exercises:

"Device Function & Flow" (Group Activity): For given scenarios, identify the input, processing, and output devices involved.

"Storage Solution Challenge" (Discussion): Recommend the most suitable storage type for various data needs.

Practical Exploration: Handle and discuss the physical characteristics of different storage media.

Possible Homework:

Table Creation: Create a table of 3 direct data entry and 3 output devices with their uses, advantages, and disadvantages.

Short Essay: Explain why an SSD is preferred over an HDD for a laptop's primary storage.

Week 4: Networks and the Internet (Ch 4.1, 10.2)

Link to Class Presentation

Theory Concepts & Definitions:

Network Types: Define and differentiate between LAN, WLAN, and WAN.

Network Devices: Explain the purpose of routers, switches, and NICs.

Cloud Computing: Describe the uses and issues related to cloud computing.

Internet, Intranet, Extranet: Differentiate between these network types and their access levels.

In-Class Exercises:

"Network Diagramming" (Individual/Pair Activity): Draw a simple home network diagram, labeling devices and connections.

"Cloud Computing Pros & Cons" (Class Debate): Debate the advantages and disadvantages of cloud storage.

"Network Scenario Matching" (Group Activity): Identify the most appropriate network type for various scenarios.

Possible Homework:

Research & Application: Find an example of a popular online service that relies on cloud computing and describe how they use it.

Define in your own words: "What is the primary difference between a hub and a switch?"

Week 5: Network Issues & Communication (Ch 4.2, 10.1, 10.2 cont.)

Link to Class Presentation

Theory Concepts & Definitions:

Security Issues: Understand common security issues related to data transfer (passwords, anti-malware).

Electronic Conferencing: Describe different methods like video, audio, and web conferencing.

Email Communication: Explain the characteristics and uses of email, including constraints and spam prevention.

Internet Functionalities & Risks: Identify key internet components (ISP, URL, web browser) and understand protocols (HTTP, HTTPS, FTP, SSL) and internet use risks.

In-Class Exercises:

"Strong Password Workshop" (Practical/Discussion): Practice creating strong passwords and discuss best practices.

"Email Etiquette Scenario" (Role Play/Discussion): Discuss or role-play appropriate email etiquette for different situations.

"Website Reliability Check" (Practical): Evaluate the reliability of different websites based on various criteria.

Possible Homework:

Reflection: Explain why it is important to use HTTPS for online banking.

Mini-Report: Research and explain one method to prevent computer viruses or malware.

Week 6: The Effects of Using IT & ICT Applications (Ch 5, 6, 9.1)

Link to Class Presentation

Theory Concepts & Definitions:

Effects of Microprocessors: Analyze the positive and negative effects of microprocessor-controlled devices on daily life and transport.

Health Problems & Prevention: Identify potential health problems from IT use (RSI, back problems) and strategies for prevention.

ICT Applications: Describe the uses of various ICT applications (communication, modelling, etc.).

Audience Appreciation: Understand the importance of tailoring ICT solutions for a specific audience.

In-Class Exercises:

"Ergonomics Check-up" (Practical): Assess your own workstation setup and identify areas for improvement.

"ICT Application Scenarios" (Group Activity): Brainstorm and present a unique, real-world example of how ICT is used in a specific area.

"Audience Tailoring" (Discussion): Discuss how an instruction manual would change for different audiences.

Possible Homework:

Reflection: Choose one microprocessor-controlled device and describe one positive and one negative impact.

Design a Healthy Workstation: Draw a diagram or write a detailed description of an ideal ergonomic workstation.

Week 7: Copyright & File Management Basics (Ch 9.2, 11.1)

Link to Class Presentation

Theory Concepts & Definitions:

Copyright: Explain the need for copyright legislation and methods used to prevent infringement.

File Management: Understand the importance of hierarchical folder structures and appropriate file names.

File Formats: Identify common file formats (.docx, .pdf, .jpg, etc.) and their uses.

In-Class Exercises:

"Copyright Dilemmas" (Group Discussion): Discuss the ethical and legal implications of various copyright scenarios.

File Management Practical (Hands-on): Create a folder structure, save and rename files, and explore file types.

Possible Homework:

Research: Find a real-world example of copyright infringement.

Practice: Create a new folder structure for organizing your school work and take a screenshot.

Week 8: Document Production - Page Layout & Basic Formatting (Ch 17.1-17.2)

Link to Class Presentation

Theory Concepts & Definitions:

Page Layout: Understand and apply page layout options like page size, orientation, margins, columns, and breaks.

Text Formatting: Format text using line spacing, paragraph spacing, tabulation, indentation, and text enhancements (bold, italic, etc.).

In-Class Exercises:

Word Practical - "Newsletter Section" (Hands-on): Practice setting custom margins, changing orientation, and creating a two-column layout.

Word Practical - "Poem Formatting" (Hands-on): Apply different line spacing and text enhancements to a poem.

Word Practical - "Tabulated List": Use decimal tabs to align a list of items and prices.

Possible Homework:

Practice: Recreate a short online article in Word and experiment with different spacing and text case.

Reflection: Explain why section breaks are important for documents with different page layouts.

Week 9: Document Production - Lists, Find & Replace, Navigation (Ch 17.2 cont., 17.3, 17.4)

Link to Class Presentation

Theory Concepts & Definitions:

Lists: Create and edit bulleted and numbered lists, including multi-level lists.

Find and Replace: Utilize this feature for quick edits and corrections.

Navigation: Add and delete bookmarks and hyperlinks for internal and external navigation.

In-Class Exercises:

Word Practical - "Project Outline & To-Do List" (Hands-on): Create a multi-level numbered list for an outline and a bulleted list for a to-do list.

Word Practical - "Document Clean-up" (Hands-on): Use Find and Replace to correct inconsistencies in a provided document.

Word Practical - "Interactive Mini-Report" (Hands-on): Add bookmarks and hyperlinks for internal and external navigation.

Possible Homework:

Practice: Create a document outlining a meal preparation, using lists and a hyperlink.

Challenge: Use Find and Replace to change the word "computer" to "ICT device" where it is not capitalized.

Week 10: Document Production - Tables, Headers & Footers (Ch 13.2, 13.3)

Link to Class Presentation

Theory Concepts & Definitions:

Tables: Create and edit tables, including formatting options like merging cells and text wrap.

Headers & Footers: Use headers and footers to place consistent information and automated objects like page numbers and dates.

In-Class Exercises:

Word Practical - "Class Schedule Table" (Hands-on): Create a weekly class schedule table with merged cells and adjusted row/column sizes.

Word Practical - "Product Comparison Table" (Hands-on): Create and format a table to display product comparison data.

Word Practical - "Professional Document Header/Footer" (Hands-on): Add a header with the document title and a footer with automated page numbers and date.

Possible Homework:

Table Design: Design a table to display information about 5 input devices, applying appropriate formatting.

Header/Footer Practice: Add a header with your name and class, and a footer with the automated file name.

Week 11: Document Production - Styles & Proofing & Comprehensive Project Work (Ch 14, 15, and all previous Word skills)

Link to Class Presentation

Theory Concepts & Definitions:

Styles: Create, modify, and apply styles for consistent document presentation.

Proofing: Use automated tools and manual proofreading to reduce errors.

Validation vs. Verification: Differentiate between these concepts and their importance in data entry.

Comprehensive Word Project: Apply a wide range of MS Word skills in a major project.

In-Class Exercises:

Word Practical - "Style Master" (Hands-on): Create and apply new custom paragraph styles.

Word Practical - "Proofreading Challenge" (Hands-on): Use spell/grammar check and manual proofreading to correct errors.

Comprehensive Word Project (Dedicated Work Time): Work on the major graded project for Term 1.

Possible Homework:

Project Completion: Finalize and submit the Comprehensive Word Project.

Study: Review all Term 1 theory for the end-of-term review/quiz next week.
