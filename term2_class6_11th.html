Part 1: Setup & Ranges

  • Open: Week2_Solar_Procurement.csv.
  • Adjust Layout: Auto-fit all column widths.
  • Identify Ranges: Locate the Component Database (starts at A14) and the Shipping Rates table (starts at B21).
  • Formatting: Make Row 2 headers Bold and Blue. Apply a Thin Border to the main Order Form (A2:I12).

Part 2: Vertical Lookup (VLOOKUP)

  • Description Lookup (Column D): In cell D3, write a VLOOKUP formula to find the Item Name based on the Component Code (B3).
    • Table Array: Use the Component Database range ($A$14:$E$18). CRITICAL: Use Absolute References ($).
    • Column Index: The Item Name is in column 2.
    • Range Lookup: FALSE (Exact Match).
  • Unit Price Lookup (Column E): In cell E3, use VLOOKUP to find the Unit Price from the same database. Lock the table array!
  • Drag: Drag both formulas down to Row 12.

Part 3: Calculations & Logic (IF)

  • Total Cost (Column F): In F3, calculate Quantity * Unit Price. Drag down.
  • Budget Check (Column G): We have a rule: Orders over $50.00 require "Approval". Others are "Auto".
  • In cell G3, enter an IF statement:
    • Logical Test: Is Total Cost (F3) greater than 50?
    • Value if True: "Approval"
    • Value if False: "Auto"
  • Drag down.

Part 4: Horizontal Lookup (HLOOKUP)

  • Shipping Cost (Column I): The Shipping Class is in Column H (A, B, or C). The costs are in the horizontal table at the bottom.
  • In cell I3, write an HLOOKUP formula.
    • Lookup Value: The Shipping Class (H3).
    • Table Array: The Shipping Rates table range ($B$21:$D$22). CRITICAL: Use Absolute References ($).
    • Row Index: The cost is in row 2.
    • Range Lookup: FALSE.
  • Drag down.

Part 5: Formatting & Finalizing

  • Currency: Format Columns E, F, and I as Currency ($) with 2 decimal places.
  • Conditional Formatting: Highlight the Budget Check column (G3:G12).
    • Apply a Rule: If text contains "Approval", fill with Red background and White text.
  • Save: As Week2_Homework_YOURNAME.xlsx.
