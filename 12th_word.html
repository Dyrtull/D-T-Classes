Welcome!

This guide provides a detailed walkthrough for completing Task 2 from the Feb/March 2021 exam paper. We will be using Microsoft Word to edit the document `M21LABPRAC.rtf`. Follow each step carefully to ensure you meet all the requirements.

1 Open and Save the Document

  1. Open Microsoft Word.
  2. Go to File > Open and navigate to your source files folder.
  3. Open the file `M21LABPRAC.rtf`.
  4. Go to File > Save As.
  5. In the "Save As" dialog box, name the file M21LABWORK.
  6. From the "Save as type" dropdown menu, select Word Document (*.docx). This saves it in the format of the software you are using.
  7. Click Save.

2 Page Size and Margins

  1. Go to the Layout tab in the Word ribbon.
  2. Click on Size and select A4.
  3. Click on Orientation and select Portrait.
  4. Click on Margins and select Custom Margins... at the bottom.
  5. In the Page Setup window:
    • Set the Top margin to 3 cm.
    • Set the Bottom, Left, and Right margins to 2 cm.
  6. Click OK.

3 Header and Footer

Double-click at the top of the page to open the header area. The Header & Footer tab will appear.

Header Setup:

  1. With the header open, ensure you see three sections for left, center, and right alignment. If not, you can use tabs.
  2. Left Align: Type your name, centre number, and candidate number.
  3. Center Align: Press the `Tab` key once. Type Tawara University Science.
  4. Right Align: Press `Tab` again.
    • Go to Header & Footer > Pictures > This Device....
    • Select `M21IMAGE.jpg` and click Insert.
    • Click on the inserted image. Go to the Picture Format tab.
    • In the Size group, set the Width to 2cm. Ensure the "Lock aspect ratio" box is checked.
    • Click on Wrap Text and choose In Front of Text or Through to ensure it doesn't disrupt the header text. Position it neatly on the right margin.

Footer Setup:

  1. While in the Header & Footer view, click Go to Footer.
  2. Left Align: Go to Header & Footer > Page Number > Current Position > Plain Number.
  3. Right Align: Press the `Tab` key twice. Go to Header & Footer > Document Info > File Path. This inserts the full file name and path automatically.
  4. Double-click the main body of the document to close the Header & Footer view.

4 Correcting Paragraph Styles

Two styles have been applied incorrectly. We need to find and fix them by modifying the style definitions.

  1. Identify & Correct Style 1 (Lab-Subhead):
    • The house style requires `Lab-Subhead` to be bold and underlined. The current text is only bold.
    • On the Home tab, right-click on the `Lab-Subhead` style in the Styles gallery and select Modify....
    • Click the U (Underline) button to add underlining.
    • Click OK. All text with this style will update automatically.
  2. Identify & Correct Style 2 (Lab-Body):
    • The house style requires `Lab-Body` to have 1.5 line spacing. It is currently set to single spacing.
    • Right-click on the `Lab-Body` style and select Modify....
    • In the bottom-left corner, click Format > Paragraph....
    • Under the "Spacing" section, change "Line spacing" to 1.5 lines.
    • Click OK twice to close both windows.

5 Final Document Formatting

  1. Add Your Name: Find the subtitle "Guide produced by:" and type your name on the same line after the colon.
  2. Create Columns:
    • Place your cursor just before the subheading "Vernier Scales".
    • Go to Layout > Breaks > Continuous. This inserts a section break.
    • Now, with your cursor in the "Vernier Scales" section, go to Layout > Columns > More Columns....
    • Choose Two columns.
    • Set the Spacing to 1 cm.
    • Ensure "Apply to:" is set to This section. Click OK.
  3. Apply Bullet Style: Highlight the text from "The lab book is evidence..." down to "...is a bound A4 notebook.". On the Home tab, apply the `Lab-Bullets` style from the gallery.
  4. Format Paragraph:
    • Find and highlight the paragraph that starts "These are just some of the items...".
    • Right-click and select Paragraph....
    • Under "Indentation", set both Left and Right to 1 cm. Click OK.
    • With the paragraph still selected, go to the Home tab. In the Paragraph group, click the Borders dropdown menu and select Outside Borders.
    • Click the Shading (paint bucket) dropdown and select a light grey color.
  5. Format Link Text: Find the text `[insert link to tutorial here]`. Highlight it (including the brackets) and apply both Bold and Italic formatting.
  6. Spell Check and Proofread: Press `F7` or go to Review > Spelling & Grammar. Review the entire document for errors, widows/orphans (a single word on a new line), and ensure lists are not split across columns.
