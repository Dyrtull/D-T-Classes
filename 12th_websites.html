Tawara Construction Web Page Tutorial

A step-by-step guide to completing Task 3 from the 0417/31 Information and Communication Technology paper using Microsoft Expression Web 4.

Click here to see the necessary files for this task

2

Create the Web Page and Table Structure

First, open Microsoft Expression Web 4. Go to File > New > PHP. Save the file as m25plaster.htm in your plaster folder.

Now, you need to create the table structure. Go to Insert > Table. Set the following properties for the table:

  • Rows: 3
  • Columns: 2
  • Width: 95%
  • Alignment: Center

Modify the table to match the required layout. You will need to merge cells to achieve the structure A, B, C, D, E, F, G. The diagram shows a 3x2 table with merged cells:

  • Merge the two cells in the first row to create Cell A.
  • The second row has two cells, B and C.
  • The third row will have four cells. You can achieve this by splitting the two cells in the third row into two columns each, or by inserting a 1-row, 4-column table into the merged third row cell.
3

Insert the Logo Image

In Cell A, place the image m25logo.jpg. Go to Insert > Picture > From File and select the image. Once the image is in the cell, right-click it and choose Picture Properties. In the "Appearance" tab, set the "Width" to 100% and ensure the "Maintain aspect ratio" checkbox is selected. You can also do this in the code view by adding width="100%" to the <img> tag.

4

Add the Video

In Cell B, you need to add the video m25plaster.mp4. The simplest way is to go into the code view and add the following HTML:

<video width="100%" controls loop>
    <source src="m25plaster.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

This code will display the video, show controls, and make it loop. The text "Your browser does not support the video tag." is the automated error message for browsers that don't support the video.

5

Place the Text and Apply Style

In Cell C, open the file m25text.txt, copy the content, and paste it into the cell. Then, select the text and apply the paragraph style. In the design view, you can typically find a dropdown menu for styles (e.g., "p", "h1", "h2") on the toolbar. Select "p" for paragraph.

6

Add and Style the Footer Text

Enter the following text into the respective cells:

  • Cell D: Specialist plastering
  • Cell E: Other construction
  • Cell F: Contact us
  • Cell G: Edited by: [Your Name], [Centre Number], [Candidate Number]

Select all this text and apply the h2 style to it. You can do this in the same way you applied the paragraph style in the previous step.

7

Create the Hyperlinks

This step is crucial. You need to create three different types of links:

  • Specialist plastering: Select the text, right-click, and choose Hyperlink. In the dialog, link to the file m25spec.htm. Click the Target Frame button and select New Window (_blank).
  • Other construction: Select the text and link it to the file m25const.htm. This link should open in the same window, so you don't need to change the target frame.
  • Contact us: Select the text, right-click, and choose Hyperlink. In the dialog, choose "E-mail Address" from the left sidebar. Enter tawara.construction@cambridge.org in the email address field and More information please! in the subject field.
8

Create a New Stylesheet

Go to File > New > CSS. Add the following CSS code to the file. Remember to replace [Your Name], [Centre Number], and [Candidate Number] with your actual details.

/* Created by: [Your Name], [Centre Number], [Candidate Number] */
table {
  border-collapse: separate;
  border-style: solid;
  border-spacing: 0; /* Important: reset border-spacing to 0 for this style */
  border-color: #ffffff;
  border-width: 2px;
}

td {
  border-style: solid;
  border-color: #ffffff;
  border-width: 2px;
}

Save this file in your plaster folder with the name m25plaster2_ZZ999_9999.css, replacing the ZZ999_9999 with your centre and candidate numbers.

9

Attach the Stylesheets

Go back to your m25plaster.htm file. To attach the stylesheets, you can go to Format > CSS Styles > Attach Stylesheet or manually edit the code in the <head> section. You need to attach two stylesheets with m25plaster2.css having a higher priority.

<link rel="stylesheet" type="text/css" href="m25plaster.css">
<link rel="stylesheet" type="text/css" href="m25plaster2_ZZ999_9999.css">

The stylesheet that appears last in the code has the highest priority, so the order is correct. Make sure to replace the file name with your actual file name.

10

Final Save and Evidence

Save your web page (m25plaster.htm). Now, you need to provide the required evidence. For the HTML source, go to Code view, copy all the code, and paste it into your Evidence Document. For the browser view, save the file, then open it in a web browser. Resize the window to fit the page and show the address bar, then take a screenshot. Place the screenshot in your Evidence Document.
