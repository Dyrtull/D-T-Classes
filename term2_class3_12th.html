Digital vs Analog Input

Understanding the difference is CRITICAL to this week's success.

üîò DIGITAL (Last Week)
  • Button = ON or OFF
  • Code: digitalRead()
  • Returns: HIGH or LOW
  • Like a light switch (2 states)
üåä ANALOG (This Week!)
  • Sensor = Continuous range
  • Code: analogRead()
  • Returns: 0 to 1023
  • Like a dimmer switch (infinite states)
DIGITAL: ANALOG: ‚îÅ‚îÅ‚îÅ‚îÅ ON ‚îÅ‚îÅ‚îÅ‚îÅ‚îÅ‚îÅ 1023 (Max brightness) ‚îÅ‚îÅ‚îÅ‚îÅ‚îÅ‚îÅ 512 (Medium) _______ OFF ‚îÅ 0 (Off) Only 2 options 1024 possible values!

üå°Ô∏è GROUP 1: KY-013 Thermistor (Temperature) ‚ñº

Physical Description:

  • Pre-assembled module (black PCB)
  • 3 pins: VCC, GND, A0
  • No polarity issues (works either direction)
  • Thermistor measures temperature

Wiring (Simple!):

KY-013 Pinout: VCC (red) ‚îÄ‚îÄ> Arduino 5V GND (black) ‚îÄ‚îÄ> Arduino GND A0 (yellow) ‚îÄ‚îÄ> Arduino A0

Technical Specs:

  • Range: -50¬∞C to +110¬∞C
  • Accuracy: ¬±2¬∞C typical
  • Output voltage varies with temperature
  • Thermistor: NTC 10kŒ©

Conversion Formula:

1. Read analog value (0-1023)
2. Convert to voltage: voltage = value √ó (5.0 / 1023.0)
3. Apply thermistor formula:
   temp(¬∞C) = (voltage - 0.5) √ó 100
   (Approximation; adjust based on calibration)
üí° Why It's Different:

KY-013 is a module (pre-assembled), not a bare sensor. Makes wiring simpler and more reliable than TMP36!

üé§ GROUP 2: KY-038 Sound Sensor (Noise Level) ‚ñº

Physical Description:

  • Pre-assembled module (red PCB)
  • 3 pins: VCC, GND, A0
  • Built-in microphone detects sound
  • Adjustable sensitivity potentiometer

Wiring (Simple!):

KY-038 Pinout: VCC (red) ‚îÄ‚îÄ> Arduino 5V GND (black) ‚îÄ‚îÄ> Arduino GND A0 (yellow) ‚îÄ‚îÄ> Arduino A0

Technical Specs:

  • Frequency range: 100-8000 Hz
  • Output: 0-1023 (sound level)
  • Sensitive microphone detects ambient noise
  • Has potentiometer for calibration

Reading Sound Levels:

1. Read analog value (0-1023)
2. 0 = Silence
3. 500-700 = Normal talking
4. 800+ = Loud noise (shouting, music)
5. Adjust sensitivity using onboard dial
üí° Why It's Different:

KY-038 measures sound/noise instead of temperature. Same analog concept, DIFFERENT REAL-WORLD APPLICATION!

Example 2A: Temp Alarm (KY-013) üö®

Challenge: Make LED turn ON when temperature > 25¬∞C

int sensorPin = A0;
int ledPin = 13;

void setup() {
  Serial.begin(9600);
  pinMode(ledPin, OUTPUT);
}

void loop() {
  int raw = analogRead(sensorPin);
  float voltage = raw * (5.0 / 1023.0);
  float tempC = (voltage - 0.5) * 100.0;
  
  if (tempC > 25.0) {
    digitalWrite(ledPin, HIGH);
    Serial.println("HOT!");
  } else {
    digitalWrite(ledPin, LOW);
    Serial.println("OK");
  }
  
  delay(1000);
}

Test:

  • Hold sensor ‚Üí temp rises ‚Üí LED ON
  • Release ‚Üí temp drops ‚Üí LED OFF

Example 3: Combo Challenge (Both Groups Together!) ü§ù

Goal: Use BOTH sensors on ONE Arduino (Group 1 + Group 2 partner up!)

Wiring (Temp on A0, Sound on A1):

KY-013 (Temperature): VCC ‚îÄ‚îÄ> 5V GND ‚îÄ‚îÄ> GND A0 ‚îÄ‚îÄ> Arduino A0 KY-038 (Sound): VCC ‚îÄ‚îÄ> 5V GND ‚îÄ‚îÄ> GND A0 ‚îÄ‚îÄ> Arduino A1

Combo Code:

int tempPin = A0;
int soundPin = A1;
int ledPin = 13;

void setup() {
  Serial.begin(9600);
  pinMode(ledPin, OUTPUT);
}

void loop() {
  // Read both
  int tempRaw = analogRead(tempPin);
  float tempC = ((tempRaw * 5.0 / 1023.0) - 0.5) * 100.0;
  int soundLevel = analogRead(soundPin);
  
  Serial.print("Temp:");
  Serial.print(tempC);
  Serial.print("C | Sound:");
  Serial.println(soundLevel);
  
  // LED on if EITHER too hot OR too loud
  if (tempC > 25.0 || soundLevel > 600) {
    digitalWrite(ledPin, HIGH);
    Serial.println("ALERT!");
  } else {
    digitalWrite(ledPin, LOW);
  }
  
  delay(500);
}
üí° Key Learning:

You can read MULTIPLE sensors! Use different analog pins (A0, A1, A2...). Logic works the same: analogRead() ‚Üí compare ‚Üí decide ‚Üí act.

üìñ Vocabulary

Analog Input
A continuous range of values (0 to 1023) representing varying measurements, unlike digital signals that are only ON/OFF.
Example: A temperature sensor returns values like 312, 313, 315 ‚Äî not just 0 or 1.
analogRead()
Arduino function that reads a voltage on an analog pin (A0-A5) and converts it to a digital number between 0-1023.
Example: int value = analogRead(A0); reads pin A0 and stores result in variable 'value'.
Threshold
A decision point in code where one value divides two different outcomes. If below threshold ‚Üí one action. If above ‚Üí different action.
Example: if (temperature > 25) { alarm = ON; } ‚Äî threshold is 25¬∞C.
Sensor
A device that detects changes in physical conditions (temperature, light, motion, etc.) and converts them into electrical signals Arduino can read.
Example: TMP36 senses temperature and outputs a voltage proportional to how hot it is.
Serial Monitor
A tool in Arduino IDE that displays data sent from Arduino to your computer via USB connection, used for debugging and viewing sensor readings.
Example: Open with magnifying glass icon, set baud rate to 9600, watch sensor values scroll by.
Baud Rate
The speed of serial communication measured in bits-per-second. Both code and Serial Monitor must use the same baud rate or communication fails.
Example: Serial.begin(9600) sets baud rate to 9600. Serial Monitor must also be set to 9600.
ADC (Analog-to-Digital Converter)
Hardware inside Arduino that converts analog voltages (continuous 0-5V) into digital numbers (0-1023) that code can use.
Example: 2.5V input ‚Üí ADC converts to 512 (middle of 0-1023 range).
KY-013 Thermistor Module
A pre-assembled temperature sensor module with NTC 10kŒ© thermistor. 3 pins: VCC, GND, A0. Outputs voltage proportional to temperature.
Example: Room temperature (~22¬∞C) outputs ~1.52V. Formula: temp(¬∞C) = (voltage - 0.5) √ó 100.
KY-038 Sound Sensor Module
A pre-assembled microphone module that detects ambient sound/noise levels. 3 pins: VCC, GND, A0. Outputs analog value 0-1023 based on sound pressure.
Example: Silence = 200-300. Normal talking = 500-700. Clapping/shouting = 800+. Sensitivity adjustable with onboard potentiometer.
Module
A pre-assembled circuit board with a sensor plus necessary supporting components (resistors, capacitors, etc.). Makes wiring simpler than bare sensors.
Example: KY-013 module includes thermistor + supporting circuitry. Just wire 3 pins instead of building complex circuit.
Debugging
The process of finding and fixing errors in code or circuits. Serial Monitor is a primary debugging tool.
Example: Code doesn't work ‚Üí use Serial Monitor to print values and see where logic fails.
Calibration
Adjusting sensor values or thresholds to work correctly in your specific environment, as conditions vary between locations.
Example: Light threshold of 300 works in your classroom but needs to be 500 in a brightly-lit hallway.
