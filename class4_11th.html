Networks and the Internet

An Interactive Learning Guide

Theory Concepts & Definitions

Network Types

A network is a group of two or more devices connected together to share resources. There are three main types of networks:

  • Local Area Network (LAN): A network that covers a small geographical area, such as a single home, office, or school. All the hardware is typically owned by the organization or household.
  • Wireless Local Area Network (WLAN): A type of LAN where devices connect wirelessly using radio waves (like Wi-Fi) instead of cables.
  • Wide Area Network (WAN): A network that spans a large geographical area (over one mile), connecting multiple LANs together. The Internet is the most well-known example of a WAN.

Network Devices

These are the pieces of hardware that allow networks to function.

  • MAC(Media Access Control) address: is a unique identifier given to devices which communicate over a local area network (LAN) MAC addresses are static, they can never change. Used by switches to efficiently forward data to the intended recipient.
  • Bridge: A network bridge is a device used to connect two local area networks (LANs) together to create one larger network. Unlike a switch/hub, bridges only connect local area networks together.
  • Router: Routes data packets between different networks and assigns IP addresses to devices on a network.
  • Switch: Connects multiple devices on a network and sends data only to the specific device it's intended for, which makes the network more efficient.
  • Hub: A "dumb" device that connects multiple devices but broadcasts all data to every device, which is inefficient and a security risk.
  • Network Interface Card (NIC): A component required for a computer to connect to a network. It can be wired or wireless.

Cloud Computing

Cloud computing is the delivery of on-demand computing services—including servers, storage, databases, networking, software, analytics, and intelligence—over the Internet ("the cloud").

Advantages:

  • Data can be accessed from anywhere on any device.
  • Easily scalable to increase or decrease storage capacity.
  • Data is often backed up automatically, reducing data loss risk.
  • Security features like data encryption are often provided by the provider.

Disadvantages:

  • Requires a stable internet connection.
  • Vulnerable to security breaches and cyber-attacks.
  • Dependence on the provider's availability and reliability.
  • Can become expensive over time as usage increases.

Internet, Intranet, and Extranet

An IP (Internet Protocol) address is a unique identifier given to devices that communicate over the Internet (WAN). It makes it possible to deliver data to the right device.

There are two main versions of IP addresses:

  • IPv4: The older version, represented as four blocks of numbers (0-255) separated by dots, like 192.168.1.1. It uses 32 bits and can provide over 4 billion unique addresses.
  • IPv6: The newer version, represented as eight groups of four hexadecimal digits separated by colons, like 2001:0db8:85a3:0000:0000:8a2e:0370:7334. It uses 128 bits and can provide a nearly unlimited number of unique addresses.
  • Internet: A global network of networks accessible to everyone.
  • Intranet: A private network accessible only to employees within an organization.
  • Extranet: An extension of an intranet that allows authorized external users (like clients or partners) to access specific resources.

In-Class Exercises

Network Diagramming

Draw a simple diagram of a home network, including and labeling devices like a router, switch, computers, and a printer. Show how they are connected. You can do this on paper or using a digital tool.

Cloud Computing Pros & Cons

In a class debate, argue for or against using cloud storage. Consider the advantages and disadvantages discussed in the theory section.

Network Scenario Matching

In groups, discuss and identify the most appropriate network type (LAN, WLAN, or WAN) for the following scenarios:

  • Connecting all the computers in a single school building.
  • Connecting your phone to a wireless printer in your room.
  • Connecting a company's offices in London and New York.
