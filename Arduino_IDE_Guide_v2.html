Getting Started with Arduino IDE 2.0

A "For Dummies" step-by-step guide to connecting your board, installing libraries, and talking to your computer.

1. Connect Arduino Uno

Step A: The Physical Connection

Get your generic "Printer Cable" (USB Type-B). Plug the square end into the Arduino and the flat end into your computer.

Check: You should see a green light (labeled "ON") glow on the board immediately.

Step B: Select Board & Port

Open Arduino IDE. Look at the top dropdown menu that usually says "Select Board".

  1. Click the dropdown at the very top.
  2. If you see "Arduino Uno" detected, click it!
  3. If not, click "Select other board and port..."
  4. Search for "Uno" and select the COM port (Windows) or /dev/cu (Mac) that appears next to it.

2. Connect ESP32

First Time Setup Required!

Unlike the Uno, the ESP32 is not installed by default. You must install the "Drivers" first.

Phase 1: Add URL

Go to File > Preferences.

Paste this into "Additional Boards Manager URLs":

https://espressif.github.io/arduino-esp32/package_esp32_index.json

Click OK when done.

Phase 2: Install Board

Click the Boards Manager icon (looks like a board) on the left sidebar.

  • Type "ESP32" in the search.
  • Look for esp32 by Espressif Systems.
  • Click INSTALL.
  • Wait (it takes time!).
Phase 3: Connect

Plug in your ESP32 with a Micro-USB cable.

Warning: Many cheap cables are "Charge Only". If nothing happens, try a different cable!

Select Board: DOIT ESP32 DEVKIT V1 (This is the most common one).

3. How to Upload Code

Step A: Verify

Click the Checkmark icon at the top left first.

This checks your code for spelling errors (Compilation). It does NOT send code to the board yet.

> Done compiling.
Sketch uses 924 bytes (2%)...

Step B: Upload

Click the Right Arrow icon next to the checkmark.

This compiles the code again and actually sends it down the USB cable to your Arduino.

> Done uploading.

ESP32 Troubleshooting Tip

If the upload gets stuck at Connecting........_____.....:

  1. Hold down the BOOT button on your ESP32 board.
  2. Wait for the text to change to "Writing".
  3. Release the button.

4. How to Install a Library

Need to control a servo, a sensor, or a screen? You need a library. It's like downloading an app for your phone.

The "Library Manager" Icon

  1. 1

    Open the Manager

    On the left sidebar, click the icon that looks like a stack of books. Or go to main menu: Sketch > Include Library > Manage Libraries...

  2. 2

    Search

    Type the name of the part you are using (e.g., "DHT11", "Adafruit NeoPixel", "Servo").

  3. 3

    Install

    Find the one you want in the list and click the INSTALL button. If it asks to install "dependencies" (helper files), say Install All.

5. Using Serial Monitor

This is how the Arduino sends text messages back to your computer.

1. The Code Requirement

You MUST have this line inside your void setup():

Serial.begin(9600);

2. Opening the Monitor

Look at the top right corner of the IDE. Click the icon that looks like a magnifying glass.

<- Click this button

3. The Baud Rate Rule

The number in the bottom right corner of the Serial Monitor (e.g., 9600 baud) MUST match the number in your code (e.g., Serial.begin(9600)).

If you see weird symbols like "⸮⸮⸮", your baud rate is wrong!
