Task 3: Step-by-Step Web Page Creation Guide (Expression Web 4 GUI)

These steps walk you through creating the racing.htm file using the graphical user interface (GUI) features of Microsoft Expression Web 4. The final HTML source is required for the Evidence Document in Step 15.

Web Page Setup (Steps 4-5)

  • Create & Save: Go to File > New > Page, choose HTML, and save immediately as racing.htm in the racing folder.
  • Set Title: In Design view, Right-click anywhere on page > Page Properties. Set the Title field to: Tawara Racing.

Table Structure (Steps 6-7)

Always be in Design view for visual editing.

  • Insert Table: Go to Table > Insert Table. Set Rows: 4, Columns: 2, Width: 90 percent, Border thickness: 1, and Alignment: Left.
  • Merge Cells:
    • Select the first row (drag across both cells) and Right-click > Merge Cells (Cell A).
    • Select the third row (both cells) and Right-click > Merge Cells (Cell D).
    • Select the fourth row (both cells) and Right-click > Merge Cells (Cell E).

Content Insertion (Steps 8-12)

  • Insert Banner (Cell A): Click in Cell A. Go to Insert > Picture > From File and select TRbanner.jpg.
  • Insert Video (Cell B): Click in Cell B. Go to Insert > Media > Video. In the dialog, set Source: TRvideo.mp4 and check the "Show controls" and "Loop" boxes.
  • Insert Text (Cell C): Paste the text from TRtext.txt. Select all text and set its Style dropdown (in the formatting toolbar) to Heading 3 (h3).
  • Insert Navigation (Cell D): Type the required navigation text. Select all text and set its Style to Heading 2 (h2).
  • Create Links (Cell D):
    • Select text "exciting developments": Go to Insert > Hyperlink. Set Link to: TRdevelop.htm and Target: New window (_blank).
    • Select text "job opportunities": Go to Insert > Hyperlink. Set Link to: mailto:Tawara.Racing@cambridge.org?subject=Job opportunities.
  • Insert Footer (Cell E): Type "Web page created by:" then press Enter for a new line and type your Name, Centre Number, and Candidate Number. Select all text and set its Style to Heading 2 (h2).

Styling and Documentation (Steps 13-15)

  • Attach Stylesheet: Go to Format > CSS Styles and click the "Attach Style Sheet" button. Browse to TRracing.css and click OK.
  • Modify Stylesheet: In the CSS Styles panel, double-click TRracing.css to open it. Find the table style and change border-collapse: collapse; to border-collapse: separate;. Save the stylesheet.
  • Document: Save racing.htm. Go to View > Code view, Select All, and Copy the HTML source code to your Evidence Document.

Task 5: Detailed Spreadsheet Analysis (Excel Formulas)

These are the key formulas required for the Excel analysis section of the task. They should be entered into the specified cells to achieve the results needed for printing.

Step 12: Calculate Total Lap Time (Cell F14)

Converts minutes (C), seconds (D), and thousandths (E) into a total time in seconds.

=60*C14+D14+E14/1000

Replicate this formula down the column (e.g., F14 to F53) and format to 3 decimal places.

Step 15 - 17: Time Statistics (in column B)

  • Fastest Lap (Cell B3): =MIN(F14:F53)
  • Slowest Lap (Cell B4): =MAX(F14:F53)
  • Average Lap Time (Cell B5): =AVERAGE(F14:F53)

Step 18 - 19: Count Laps (in column B)

Counts how many laps ran with the supercharger on or off (data in range B14:B53).

  • SC ON Count (Cell B6): =COUNTIF(B14:B53,"On")
  • SC OFF Count (Cell B7): =COUNTIF(B14:B53,"Off")

Step 20 - 22: Performance Analysis (in column B)

  • Avg Time SC ON (Cell B8): =AVERAGEIF(B14:B53,"On",F14:F53)
  • Avg Time SC OFF (Cell B9): =AVERAGEIF(B14:B53,"Off",F14:F53)
  • Result Analysis (Cell B10): =IF(B8<B9,"SC success","SC failed")

The IF statement checks if the Supercharger ON average (B8) is *less* than the OFF average (B9), which indicates a faster time, hence "SC success".
