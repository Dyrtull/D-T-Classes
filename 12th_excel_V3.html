IGCSE ICT Paper 3: Spreadsheet Task Revision

Essential Revision Checklist

1. Spreadsheet Basics & Structure

A spreadsheet is software used to analyse, visualise, and manipulate data (a data model).

Terminology:
  • Cell: Single box (e.g., A1).
  • Row: Goes across (referenced by number).
  • Column: Goes down (referenced by letter).

Editing Structure: Know how to insert/delete cells, rows, and columns to manage data effectively.

Merging Cells: Combine multiple cells for creating clear headers or titles across columns.

2. Formulae & Functions

Formulae (Simple Calculations)

Formulas start with an = sign and use arithmetic operators to perform calculations on numbers or referenced cells (e.g., =A1*B2).

Arithmetic Operators (BIDMAS):
  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
  • ^ (Indices/Power of)

Remember to use brackets () to control the Order of Operations.

Functions (Pre-defined Formulae)

Functions simplify complex calculations. Know the purpose and syntax for the following:

SUM: Adds all numbers in a range.
Example: =SUM(A1:A10)
AVERAGE: Calculates the mean of a range.
Example: =AVERAGE(A1:A10)
MAX / MIN: Finds the largest/smallest number.
Example: =MAX(A1:A10)
COUNT / COUNTA: Counts cells with numbers / cells with anything.
Example: =COUNT(A1:A10)
IF: Returns one value if a condition is true, another if false.
Syntax: =IF(condition, true_value, false_value)
VLOOKUP / HLOOKUP / XLOOKUP: Performs vertical/horizontal/either lookup to find data in a table.
COUNTIF / SUMIF / AVERAGEIF: Performs the function only if a certain criterion is met.
Example: =AVERAGEIF(B2:B50, "=Mix 1", C2:C50) (Often used in past papers like 0417/32/M/J/25).
Nesting Functions: This means using one function inside another, e.g., =ROUND(AVERAGE(A1:A10), 2).

Exercise 1: Calculating Performance Metrics

Imagine a column of lap times (D2:D50) and an adjacent column indicating if the supercharger was used (E2:E50, containing "On" or "Off").

  1. Write the formula for cell A1 to display the fastest lap time from the range D2:D50.

    =MIN(D2:D50)

  2. Write the formula for cell A2 to count the number of laps where the supercharger was "On". (Referenced from 0417/31/M/J/25).

    =COUNTIF(E2:E50, "On")

  3. Write the formula for cell A3 to display "SUCCESS" if the average lap time when the supercharger was "On" (cell B8) is faster (i.e., smaller) than the average lap time when it was "Off" (cell B9), or "FAILED" if not. (Referenced from 0417/31/M/J/25).

    =IF(B8 < B9, "SUCCESS", "FAILED")

3. Cell Referencing & Lookups

Relative Reference (Default)

The cell reference changes when the formula is copied or dragged. E.g., copying =A1+B1 from C1 to C2 changes it to =A2+B2.

Absolute Reference (Fixed)

The cell reference does not change. Use the dollar sign ($) to fix the row, column, or both. This is crucial for fixing values like a tax rate or exchange rate.

Fix Both: =$A$1 (The reference will always be A1).
Fix Column: =$A1 (Column A is fixed, row changes).
Fix Row: =A$1 (Row 1 is fixed, column changes).

Named Cells / Ranges

Assigning a name (e.g., Tax_Rate) to a single cell or a range of cells (e.g., A3:C6). This simplifies complex formulas and automatically acts like an absolute reference.

Exercise 2: VLOOKUP (External File)

A company needs to look up the name and distance of a lodge using a code.

Data Setup (Similar to 0417/31/F/M/24):

  • Cell B4 contains the Lodge/Camp code (e.g., LC17).
  • Lookup_Table is an external file with data in three columns: Code (Col 1), Name (Col 2), Distance (Col 3).

  1. Write the formula for cell B6 to look up the Name of the lodge using the code in B4 and the external lookup table.

    =VLOOKUP(B4, 'Lookup_Table'!A:C, 2, FALSE)

  2. Write the formula for cell B7 to look up the Distance (column 3).

    =VLOOKUP(B4, 'Lookup_Table'!A:C, 3, FALSE)

4. Presenting and Formatting Data

Display Features

  • Displaying formulas vs. values.
  • Adjusting row height and column width.
  • Wrapping text for long headers.

Formatting

  • Enhancing Look: Text/cell colour, bold, italic, underline.
  • Numeric Data: Adjusting decimal places, applying currency symbols (e.g., Â£, $), and dealing with percentages.

Conditional Formatting (CRITICAL EXAM SKILL)

Dynamically changing the format (e.g., cell colour) of a cell based on its content.

Exercise 3: Conditional Formatting

You have calculated the monthly Balance for three projects (Elephant, Monkey, Tiger) in the range C23:N25. (Referenced from 0417/32/M/J/23).

Set up two conditional formatting rules for the entire range C23:N25:

  1. If the cell contains a value greater than or equal to 0, the cell must have a **light-green background** with **dark-blue text**.

  2. If the cell contains a value less than 0 (a loss), the cell must have **red text** on a **white background**.

Rule 1 Condition: Cell Value Is >= 0
Rule 2 Condition: Cell Value Is < 0

IMPORTANT: Always ensure you apply the rule to the correct range (e.g., C23:N25) and check that the **less than** rule is applied correctly for losses.

Page Layout

Know how to set the page orientation (Portrait/Landscape) and control the print layout (setting print areas, scaling, showing/hiding gridlines/row/column headings).

Print Checks: Always confirm:
  • Page orientation (Landscape/Portrait)
  • Row and column headings (Visible/Hidden)
  • Contents of all cells are fully visible (adjust column width if necessary)
  • Printout fits on a **single page** (adjust scaling or print area)

5. Search, Select, and Sort Data

Searching / Filtering (Data Selection)

Allows you to focus on a subset of data using criteria and logical/comparison operators:

Operators: >, <, =, >=, <=, <>, AND, OR, NOT.
Wildcards: * (any number of characters), ? (a single character).

Sorting

Arranging data based on specific criteria (ascending or descending order). You must be able to sort using a single criterion or multiple criteria (e.g., by Grade, then by Name).

Examiner Tip: When sorting by multiple criteria, ALWAYS select all relevant columns/the entire data range to ensure the data stays aligned!

6. Exam-Specific Task: Time Conversion

Converting Minutes and Seconds to Total Seconds

A very common task is converting time data (Minutes, Seconds, Thousandths) into a single Total Seconds value, typically to 3 decimal places.

Exercise 4: Time Conversion

You have data in cells C14 (Minutes), D14 (Seconds), and E14 (Thousandths of a second). You need a replicable formula in F14 to calculate the time in total seconds. (Referenced from 0417/31/M/J/25).

Conversion Rule: 1 minute = 60 seconds. Thousandths must be converted to seconds (e.g., 484 thousandths = 0.484 seconds).

  1. Write the replicable formula for cell F14 to calculate the total time in seconds.

    =(C14 * 60) + D14 + (E14 / 1000)

REPLICABLE: Since this formula will be copied down, **all references (C14, D14, E14) must be relative** (no $ signs), which they are in the answer above!
