Task 4: Step-by-Step Web Page Creation Guide (Expression Web 4 GUI)

These steps guide you through creating the j25racing.htm file. Pay close attention to the video embedding (Step 21), which requires a manual switch to Code View.

Web Page Setup & Table Structure (Steps 19-20)

  • Create & Save: Go to File > New > Page, choose HTML, and save immediately as j25racing.htm in the TR folder.
  • Insert Table (Step 19): Go to Table > Insert Table. Set Rows: 4, Columns: 2, Width: 85 percent, Border thickness: 0, and Alignment: Center.
  • Merge Cells (Step 19): Select the first row (both cells) and Right-click > Merge Cells (Cell A). Leave the other three rows separate.
  • Set Cell C Dimensions (Step 20): Right-click cell C (right side of Row 2) > Tag Properties. In the Style field, type: width:480px; height:480px;

Video and Content Insertion (Steps 21-23)

  • Insert Video (Step 21 - CRITICAL):
    1. Click in cell C. Go to View > Code view.
    2. Locate the <td> tag for cell C.
    3. Type this exact code between the tags: 
      <video controls loop>
    <source src="j25racing.mp4" type="video/mp4">
    Your browser does not support this video type.
</video>
    4. Return to Design View.
  • Insert Text (Cell B - Step 22): Paste the text from j25text.txt. Select all text and set its Style dropdown to Heading 3 (h3).
  • Add Navigation & Credits (Steps 23): Type "New technologies" (D), "Contact us" (E), "Designed by:" (F), and your details (G). Select all text in cells D, E, F, and G and set the Style to Heading 2 (h2).

Linking & Styling (Steps 24-26)

  • Create Links (Step 24):
    • Select "New technologies" (D): Insert > Hyperlink. Set Link to: j25tech.htm.
    • Select "Contact us" (E): Insert > Hyperlink. Set Link to: mailto:Tawara.Racing@cambridge.org?subject=Tell me more!.
  • Attach & Modify Stylesheet (Step 25): Go to Format > CSS Styles. Attach j25racing.css. Then, double-click the CSS file and add margin-left: auto; margin-right: auto; to the table style rule to center it. Save the CSS file.
  • Apply Class (Step 26): Right-click text in cell F ("Designed by:") > Tag Properties. Set the class attribute value to: right.

Task 2: Detailed Spreadsheet Analysis (Excel Formulas)

These are the key formulas required for the Excel analysis section of the task, referencing the test times in the TTime named range (D18:D77).

General Analysis Formulas (Cells B4 - B9)

  • Fastest Test (B4): =MIN(TTime)
  • Slowest Test (B5): =MAX(TTime)
  • Average Test (B6): =AVERAGE(TTime)
  • Avg Mix 1 (B7): =AVERAGEIF(B18:B77,1,TTime)
  • Avg Mix 2 (B8): =AVERAGEIF(B18:B77,2,TTime)
  • Avg Mix 3 (B9): =AVERAGEIF(B18:B77,3,TTime)

Step 10: Wind Acceptability Check (Cell F18)

This determines if the wind adjustment (Column E) is less than +2. Replicate from F18 to F77.

=IF(E18<2,"Yes","No")

Wind-Adjusted Analysis (Cells B12 - B15)

These calculate averages only for tests where wind is "Yes" (acceptable).

  • Avg Mix 1 Adj (B12): =AVERAGEIF(F18:F37,"Yes",D18:D37)
  • Avg Mix 2 Adj (B13): =AVERAGEIF(F38:F57,"Yes",D38:D57)
  • Avg Mix 3 Adj (B14): =AVERAGEIF(F58:F77,"Yes",D58:D77)
  • Best Mix (B15): =IF(MIN(B12:B14)=B12,3.2%,IF(MIN(B12:B14)=B13,3.4%,3.6%))

Note: Best Mix formula uses a nested IF to return the mix percentage (3.2%, 3.4%, or 3.6%) corresponding to the minimum average time.
