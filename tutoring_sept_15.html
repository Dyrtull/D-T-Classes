Initial Setup and Formatting

These steps prepare your spreadsheet for the calculations. Follow the instructions to get your file ready.

  1. Open the `n24hits.csv` file in your spreadsheet software.
  2. Merge cells `A1` to `Y1`. Then, apply the following formatting: bold, center-aligned, and a font size of 24pt.
  3. Merge cells `A3` to `C3`. Right-align the text.
  4. Format the values in cells `E4` to `F6` to display with 2 decimal places.

Key Calculations (Steps 5-7)

This section focuses on placing three automated functions to calculate key metrics from the data. Click on each instruction to view the exact formula you need to use.

Average Hourly Hits

Place a function in cell B8 to calculate the average hourly number of hits for the `n24web.htm` web page.

Actual Formula:

=SUM(F8:Y8)/24

Count Pages with High Total Hits

Place a function in cell B9 to count the number of web pages with a total number of hits greater than 1000.

Actual Formula:

=COUNTIF(B8:B31, ">1000")

Count Pages with High Average Hits

Place a function in cell B10 to count the number of web pages with an average hourly number of hits greater than 30.

Actual Formula:

=COUNTIF(C8:C31, ">30")

Conditional Logic & Lookups (Steps 8-10)

These steps require using logical functions to determine traffic levels and calculate advertising revenue. Pay close attention to the conditions.

Determine Traffic Level

In cell D8, place a formula to display the traffic level for the `n24web.htm` web page.

The text should be "High" if the average hourly number of hits (cell B8) is greater than 400.

It should be "Medium" if the value is greater than 250.

Otherwise, it should be "Low".

Actual Formula:

=IF(C8>400, "High", IF(C8>250, "Medium", "Low"))

Calculate Advertising Revenue

In cell E8, place a formula to calculate advertising revenue for the `n24web.htm` web page.

Formula: Total number of hits * the rate from the threshold table.

The rate depends on the traffic level you just calculated (High, Medium, or Low).

Actual Formula:

=B8*VLOOKUP(D8, $D$4:$E$6, 2, FALSE)

Traffic Level for Highest Hits

In cell D9, place a formula to display the traffic level for the web page with the highest total number of hits. You will need to use a lookup function.

Actual Formula:

=VLOOKUP(MAX(B:B), A:D, 4, FALSE)

Final Adjustments (Steps 11-14)

These steps ensure your spreadsheet is ready to be printed and submitted for review.

  1. Format all values representing currency to display with the currency symbol and 2 decimal places.
  2. Adjust all column widths to ensure that all text and data is fully visible without being cut off.
  3. Save your spreadsheet.
  4. Print one copy of the spreadsheet showing the formulas, in landscape orientation.
  5. Print a second copy of the spreadsheet showing the values, in portrait orientation.
